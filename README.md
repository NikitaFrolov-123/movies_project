Консольное приложения списка фильмов

## Возможность
- Добавление фильма
- Отмечать просмотр
- Поиск фильмов по году

## Требования
- Python 3.8+ (рекомендуется).
- Если в проекте есть requirements.txt, установите `pip install -r requirements.txt`.

## Быстрый старт
1. Убедитесь, что установлен Python 3.
2. Запустите приложение:

```bash
python app.py
```

## Тесты
Для запуска тестов используйте:

```bash
pytest
```

Установка pytest чтобы тестировать:

```bash
pip install pytest
```
